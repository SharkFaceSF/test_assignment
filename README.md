# API Организационной структуры

Сервис для управления подразделениями и сотрудниками организации. Реализован на FastAPI + SQLAlchemy (асинхронно) + PostgreSQL.

## Функционал
- Защита от создания циклов в иерархии (департамент не может стать дочерним к своему поддереву).
- Уникальность имен подразделений на одном уровне вложенности.
- Каскадное удаление (cascade) или перенос сотрудников (reassign).
- Автоматическая очистка строк от пробелов по краям (trimming).
